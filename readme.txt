「秀丸エディタ」で「iTunes」を操作するマクロ

秀丸からiTunesを操作するマクロです。

【出来ること】
	・再生／一時停止
	・次の曲
	・前の曲
	・ミュート
	・音量の調節
	・シャッフル
	・リピート
	・再生中の曲の詳細
	・プレイリストから曲を選択して再生

	再生する操作だけできます。

＊インストール
	・このマクロを秀丸のマクロディレクトリへコピーします。
    ・このマクロをキーアサインしてください。
    ・アサインしたキーを押すとiTunesを操作するメニューがポップアップします。

＊動作環境
	・秀丸 ver8 以降のみで動作します。（COMの関数を使用しているため）
	・「田楽(でんがく)ＤＬＬ」v2.00以降が必要。
	・iTunes ver10.1.1.4で動作を確認。
